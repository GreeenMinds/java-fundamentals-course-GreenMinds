# Fundamentos de Java para Principiantes

## Resumen del Curso

Este curso introduce a estudiantes de secundaria a los fundamentos de la programación utilizando Java, incluyendo conceptos básicos de Programación Orientada a Objetos (POO).

**Duración total:** ~60 minutos

**Público objetivo:** Estudiantes de 12 a 17 años sin experiencia previa en programación.

**Prerrequisitos:** Ninguno.

**Herramientas necesarias:** Solo un navegador web (Chrome, Edge, Firefox o Safari).

**Repositorio de Código Fuente:**  
[https://github.com/GreeenMinds/java-fundamentals-course-GreenMinds](https://github.com/GreeenMinds/java-fundamentals-course-GreenMinds)
